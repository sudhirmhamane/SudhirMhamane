## 📌 Features

- 💼 Projects section with live demo and GitHub links
- 📜 Certifications with images and details
- 🛠 Skills grid with icons and categories
- 🌙 Dark background theme for elegant visual flow

## 🧩 Technologies Used

- **React.js**
- **Tailwind CSS**
- **Framer Motion**
- **React Icons**
- **Netlify** (for deployment)

---

📦 Installation & Running Locally

- git clone https://github.com/sudhirmhamane/sudhirmhamane.git
- cd sudhirmhamane
- npm install
- npm run dev

---

🚀 Deployment
This project is deployed using Netlify:

- Connect GitHub repo to Netlify.

- Set build command:
npm run build

- Set publish directory:
dist
